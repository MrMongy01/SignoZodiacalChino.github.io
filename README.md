# Documento Técnico: Calculadora de Signo Zodiacal Chino

**Materia:** Comunicación en Ingeniería  
**Equipo:** Equipo 4
**Fecha:** 17 de febrero de 2026  
**Programa elegido:** Calculadora web de signo zodiacal chino (basada en año de nacimiento)

## Introducción
Este documento presenta una aplicación web sencilla que calcula el signo zodiacal chino y su elemento asociado según el año de nacimiento del usuario.  
Fue desarrollada como ejemplo para demostrar HTML, CSS y JavaScript básico.

## Descripción del programa
- Tipo: Aplicación web estática (sin backend)  
- Tecnologías: HTML5, CSS3, JavaScript vanilla  
- Funcionalidad principal: Ingresar año → calcular signo + elemento + descripción + imagen

## Funcionalidades principales
- Validación básica del año (1900–actual)  
- Cálculo de signo chino (ciclo de 12 años)  
- Determinación del elemento (Metal, Agua, Madera, Fuego, Tierra) según último dígito  
- Muestra descripción, elemento e imagen representativa  
- Interfaz simple con formulario
